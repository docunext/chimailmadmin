Chi Mail Madmin
===============

Pronounciation
--------------
"Cheemail Madmin"


Summary
-------
A web based email management system. It can be used with: 

* Postfix
* Courier
* Dovecot
* Spamassassin
* DSpam

Requirements
------------

* Nexista.org (PHP) / Aortica.com (Perl) / NeoToyz.com (Ruby) ...
* MySQL


Installation
------------

Start with Nexista and MySQL, then use Subversion to get the current source.
